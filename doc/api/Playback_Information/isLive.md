# isLive

## Description

Returns `true` if the content is a "live" content (e.g. a live TV Channel). `false`
otherwise.

Also `false` if no content is loaded yet.

#### Example

```js
if (player.isLive()) {
  console.log("We're playing a live content");
}
```

## Syntax

```js
const isLive = player.isLive();
```

- **return value** `boolean`
